# Nova Voice User Manual

**Current V1 Build**  
**Spectral Refinement Engine**

---

## Introduction

Nova Voice is a dynamic spectral smoothing processor for vocals, instruments, and masters. It is designed to reduce harshness while preserving clarity, body, and musical detail.

The interface is intentionally fast:
- five main controls: `SMOOTH`, `FOCUS`, `AIR PRESERVE`, `BODY`, `OUTPUT`
- five preset buttons: `UNIVERSAL SMOOTH`, `VOCAL`, `DRUMS`, `GUITAR`, `MASTER`
- one `MAGIC` enhancement toggle
- a live spectral analyzer display

---

## Quick Start

1. Load `UNIVERSAL SMOOTH`
2. Raise `SMOOTH` until harshness relaxes
3. Use `FOCUS` to target where smoothing feels strongest
4. Add back openness with `AIR PRESERVE`
5. Recover weight with `BODY`
6. Level-match with `OUTPUT`
7. Toggle `MAGIC` on/off to choose the final character

---

## Controls

| Control | What you hear | What it changes |
|---|---|---|
| `SMOOTH` | Overall harshness reduction amount | Sets global smoothing intensity |
| `FOCUS` | Where smoothing energy is concentrated | Shifts reduction emphasis across the spectrum |
| `AIR PRESERVE` | Retains brightness and breath detail | Protects upper-frequency content while smoothing |
| `BODY` | Restores fullness and density | Supports low-mid and body presence during smoothing |
| `OUTPUT` | Final level trim | Applies post-processing gain |

---

## Magic Toggle

`MAGIC` changes the behavior profile of the smoothing engine and display response.

- ON: smoother, more polished, more glued behavior
- OFF: cleaner, drier, and more neutral behavior

Use it as a final tone decision after setting the main controls.

---

## Presets

| Preset | Typical use |
|---|---|
| `UNIVERSAL SMOOTH` | Balanced starting point for most sources |
| `VOCAL` | Voice-forward smoothing with clarity retention |
| `DRUMS` | Tighter transient handling with controlled edge |
| `GUITAR` | Presence control while preserving articulation |
| `MASTER` | Gentle broad smoothing for bus or full mix |

Presets are intended as starting points. Fine-tune with the five controls to fit the source.

---

## Spectral Analyzer

The analyzer shows real-time activity for:
- incoming spectral energy
- detected problem zones
- applied reduction behavior

It is a visual feedback tool to help tune settings faster. It does not generate sound by itself.

---

## Practical Tips

### Vocals
- Start with `VOCAL` or `UNIVERSAL SMOOTH`
- Increase `SMOOTH` until sibilance and edge soften
- Raise `AIR PRESERVE` if the result gets too dark

### Drums
- Start with `DRUMS`
- Keep `BODY` moderate to avoid low-mid buildup
- Use `OUTPUT` to level-match before bypass comparison

### Master Bus
- Start with `MASTER`
- Use small moves on `SMOOTH`
- Toggle `MAGIC` to choose polish vs transparency

---

## Summary

Nova Voice gives fast, musical spectral control with a small set of macro decisions:
- `SMOOTH` controls amount
- `FOCUS`, `AIR PRESERVE`, and `BODY` shape tone balance
- `OUTPUT` handles level match
- `MAGIC` changes behavior flavor
- presets accelerate workflow
- analyzer provides visual confirmation

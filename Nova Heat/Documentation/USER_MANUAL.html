These buttons are real DSP mode changes, not cosmetic toggles.

MAGIC Button

MAGIC is a finishing layer. When enabled it adds extra upper-mid focus, smoother top control, a little more dynamic sensitivity, richer saturation behavior, and a touch more stereo polish.

MAGIC changes the backend processing, but it does not automatically move the main knobs by itself.

Preset Buttons

The preset buttons recall complete starting points for common sources: VOCAL, DRUMS, BASS, and MASTER. They change the actual controls and may also toggle MAGIC.

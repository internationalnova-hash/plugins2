# Nova Level

A fresh audio plugin based on the Nova Comp UI/logic, using the v3-test.html as the UI foundation.

## Features
- Modern WebView UI (from v3-test.html)
- DSP and logic structure based on Nova Comp

## Getting Started
- Build using the provided CMakeLists.txt
- UI is located in Design/v1-test.html
- Source code in Source/

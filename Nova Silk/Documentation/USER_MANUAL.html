Magic And Presets

MAGIC changes Silk's smoothing behavior profile. ON feels more polished and glued, OFF is drier and more neutral.

Preset buttons provide starting points: UNIVERSAL SMOOTH, VOCAL, DRUMS, GUITAR, MASTER.

Best workflow: pick a preset first, then fine-tune controls and level-match with OUTPUT.

Spectral Analyzer

The analyzer displays incoming energy, detected problem zones, and reduction behavior in real time.

It is visual guidance for tuning and does not directly produce audio by itself.

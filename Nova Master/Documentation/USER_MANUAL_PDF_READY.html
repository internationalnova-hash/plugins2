Nova Master User Manual

Final Mix Processor

Nova Master is a premium finishing/mastering plugin built for fast polish, reliable tone shaping, and release-ready output confidence.

Current UI

Top bar preset browser, five shaping knobs, mode row, MIX and OUTPUT GAIN utility controls, FINISH stage, and integrated analysis/metering panels.

Parameter Ranges

ParameterRangeFunction
TONE0.0 to 10.0Broad dark/bright tonal contour.
GLUE0.0 to 10.0Cohesion and density behavior.
WEIGHT0.0 to 10.0Low-end body/foundation.
AIR0.0 to 10.0Top-end openness and polish.
WIDTH0.0 to 10.0Stereo spread and side behavior.
MIX0% to 100%Wet/dry finishing blend.
OUTPUT GAIN-12 dB to +12 dBFinal output trim.

Modes

MASTER BUS, MUSIC, STREAMING, BROADCAST, and REFERENCE are active voicing modes.

Preset Browser

Compact top-right browser with search and categories: CORE, GENRE, FIXES, and NOVA. Presets apply full parameter+mode mappings instantly.

FINISH

FINISH enables the premium finalizer stage with enhanced polish behavior, including click-time pulse, brief UI focus dim, and short stereo tighten-release motion.

Meter Views

OUT shows output-level behavior. LU shows loudness-style behavior for finishing decisions.

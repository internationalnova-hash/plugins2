Current UI Overview

  • Top bar: Presets dropdown + preset previous/next buttons
  • Main shaping: TONE, GLUE, WEIGHT, AIR, WIDTH
  • Utility: MIX, OUTPUT GAIN, FINISH
  • Meters/analysis: waveform, loudness, dynamic range, stereo image, output meter
  • Mode row: MASTER BUS, MUSIC, STREAMING, BROADCAST, REFERENCE

Preset Browser

The top-right Presets control opens a compact browser with search and categories:

  • CORE
  • GENRE
  • FIXES
  • NOVA

Active row uses gold highlight, hover rows use subtle purple/gold glow, and previous/next buttons cycle presets quickly.

FINISH

FINISH engages the premium finalizer stage with stronger polish behavior.

  • quick glow ramp on click
  • brief UI dim for focus
  • one-shot meter pulse
  • brief stereo tighten-then-release motion

Meter Views

  • OUT: output level behavior
  • LU: loudness-style behavior view

Output Status States

The status line above FINISH updates in real time as a quick output confidence guide.

StateMeaningTypical Trigger
Headroom SafeNormal safe operating state.Default when no stronger condition is flagged.
Streaming SafeTrue peak is in a streaming-safe zone.True peak at or below -1.0 dBTP.
OptimizedLoudness and peak behavior align with practical targets.Integrated loudness in range while true peak remains streaming-safe.
Watch CeilingCeiling risk warning.True peak rises above about -0.2 dBTP.
These are guidance states for workflow speed; always confirm final decisions with listening and full metering checks.
